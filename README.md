# Artists Manager — React 19 + Vite

Practice immutable array updates in React state: add, remove, replace, insert, transform, reverse, and update nested objects — including an example with **Immer**.

## Quick start

```bash
# 1) Install deps
npm install

# 2) Start dev server
npm run dev

# 3) Open the URL Vite prints (usually http://localhost:5173)
```

## Scripts
- `npm run dev` — start Vite dev server
- `npm run build` — production build to `dist/`
- `npm run preview` — preview the production build

## What to explore
- `src/App.jsx` — all the array update techniques
- `src/styles.css` — minimal, modern UI styling
